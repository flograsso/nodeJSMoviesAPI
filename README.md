# Movies API Node JS

## Puesta en marcha

1. Instalar dependencias (algunos paquetes requieren de sudo para instalarse) con : 
```
sudo npm install  
```

2. Correr la aplicacion con:
```
npm start
```

3. Correr los tests con:
```
npm test
```
